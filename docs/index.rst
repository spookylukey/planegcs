planegcs documentation
======================

**planegcs** provides Python bindings for FreeCAD's PlaneGCS 2D geometric
constraint solver. Define geometry (points, lines, circles, arcs, ellipses)
and constraints (distance, angle, tangent, parallel, etc.), then solve to
find a configuration satisfying all constraints.
